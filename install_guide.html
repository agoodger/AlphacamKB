Alphacam Knowledge Base

Installation Guide

Prerequisites

Python3.10 or later
OSWindows 10 / 11
Other dependenciesNone (stdlib only)
Disk space~150 MB
1

Copy the AlphacamKB folder

Copy the entire AlphacamKB folder to the target PC (e.g. to C:\AlphacamKB or anywhere you like).

Is Python installed?

Open a command prompt and type: python --version

YES — Python 3.10+
Skip to Step 3
NO — not found
Go to Step 2
2

Install Python (if needed)

Download from python.org/downloads

IMPORTANT: On the first installer screen, tick "Add Python to PATH" before clicking Install Now.
3

Run the installer

Open the AlphacamKB folder and double-click install.bat.

This checks Python is available and creates a Desktop shortcut. No admin rights needed.

Done!

Double-click "Alphacam Knowledge Base" on the Desktop.
The browser opens automatically to http://localhost:8080

What's in the folder

AlphacamKB/
  install.bat           -- run once to set up
  launch.bat           -- starts the server
  db_server.py         -- Python web server
  knowledge_base.db    -- article database
  ui/                  -- web interface
  kb_images/           -- extracted images
  pdfs/                -- source PDFs (Open PDF links)
  install_guide.html   -- this guide
